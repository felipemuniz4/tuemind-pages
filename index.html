Política de Privacidade — TubeMind

Última atualização: julho de 2026

O TubeMind é uma ferramenta pessoal de automação de produção e postagem de vídeos, operada por uma única pessoa para uso próprio (não é um serviço oferecido a terceiros). Esta página descreve como os dados acessados através da integração com a API do TikTok são tratados.

Quais dados são acessados

Ao autorizar o TubeMind na sua conta do TikTok, a aplicação acessa apenas o necessário para publicar vídeos em seu nome (escopo video.publish). Nenhum dado de outros usuários do TikTok é coletado ou acessado.

Como os dados são usados

Os tokens de acesso ficam armazenados localmente, apenas na máquina onde o TubeMind roda, e são usados exclusivamente para publicar vídeos que o próprio operador já revisou e aprovou dentro do sistema. Nenhum dado é enviado a terceiros, vendido ou usado para qualquer finalidade além da publicação automatizada desses vídeos.

Compartilhamento de dados

Não há compartilhamento de dados com terceiros. Não há coleta de dados de visitantes desta página (sem cookies, sem analytics, sem rastreamento).

Retenção e exclusão

Os tokens de acesso podem ser revogados a qualquer momento diretamente nas configurações de aplicativos conectados da conta do TikTok, ou removendo o arquivo de token local. Ao revogar, o acesso é encerrado imediatamente.
